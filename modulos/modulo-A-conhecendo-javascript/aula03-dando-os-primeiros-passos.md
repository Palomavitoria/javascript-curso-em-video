# Aula 03 — Dando os primeiros passos

## 📌 Introdução

Nesta aula foram apresentados os primeiros passos para começar a estudar JavaScript. 
O professor mostrou as principais ferramentas que serão utilizadas durante o curso 
e deu algumas orientações para quem está iniciando na área de programação.

---

## 🛠️ Ferramentas utilizadas no curso

Durante o curso serão utilizadas algumas ferramentas essenciais para o desenvolvimento em JavaScript.

### 🌐 Google Chrome

Navegador utilizado para executar e testar os códigos JavaScript diretamente no navegador.

### 💻 Visual Studio Code (VS Code)

Editor de código utilizado para escrever e organizar os arquivos de programação.

### ⚙️ Node.js

Ferramenta que permite executar JavaScript fora do navegador, diretamente no computador.

---

## 💡 Conselhos para iniciantes

O professor também compartilhou algumas orientações importantes para quem está começando na programação:

* Ter paciência durante o aprendizado
* Praticar constantemente
* Não desistir diante das dificuldades
* Aprender passo a passo

---

## ✅ Conclusão

A aula apresentou as ferramentas que serão utilizadas durante 
o curso e trouxe orientações iniciais para 
quem está começando a estudar programação e desenvolvimento web.
