# Aula 02 — Como chegamos até aqui?

## 📌 Introdução

Nesta aula foi apresentada a **história e evolução do JavaScript**, 
explicando como a linguagem surgiu e como se tornou uma das tecnologias mais importantes da web.

Também foi abordada a diferença entre **Java e JavaScript**, além da relação entre **JavaScript e ECMAScript**.

---

## 🕰️ Surgimento do JavaScript

O JavaScript foi criado em **1995** por **Brendan Eich**, enquanto trabalhava na empresa **Netscape**.

O objetivo era criar uma linguagem que permitisse adicionar **interatividade nas páginas da web**, que até então eram basicamente estáticas.

Inicialmente a linguagem foi chamada de **Mocha**, depois passou a se chamar **LiveScript**, e posteriormente recebeu o nome **JavaScript**.

---

## ☕ Java x JavaScript

Apesar do nome parecido, **Java e JavaScript são linguagens diferentes**.

* **Java**

  * Linguagem de programação criada pela **Sun Microsystems**
  * Muito utilizada para sistemas, aplicações corporativas e aplicativos Android

* **JavaScript**

  * Linguagem criada para rodar principalmente em **navegadores**
  * Usada para criar **interatividade em páginas web**

O nome JavaScript foi adotado principalmente por **estratégia de marketing**, pois Java estava muito popular na época.

---

## 📜 ECMAScript

O **ECMAScript** é uma **padronização da linguagem JavaScript**.

Ele define as regras e especificações que os navegadores devem seguir para implementar o JavaScript.

Ou seja:

* **ECMAScript → padrão**
* **JavaScript → implementação desse padrão**

---

## 🛠️ Ferramentas e tecnologias relacionadas ao JavaScript

Com o tempo, o JavaScript evoluiu e passou a ser utilizado em diferentes áreas do desenvolvimento.

Uma das ferramentas importantes citadas foi:

* **Node.js**

O **Node.js** permite executar JavaScript **fora do navegador**, possibilitando o desenvolvimento de **aplicações no lado do servidor**.
