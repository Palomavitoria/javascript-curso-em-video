# Aula 01 — O que o JavaScript é capaz de fazer?

## 📌 Introdução

Nesta aula inicial, o professor apresentou o papel do 
**JavaScript no desenvolvimento web**, 
explicando como as páginas da internet são construídas 
e qual a função de cada tecnologia utilizada em um site.

Também foi explicado o funcionamento da web através do modelo **Cliente x Servidor**.

---

## 🌐 Modelo Client x Server

A web funciona através da comunicação entre **cliente** e **servidor**.

* **Cliente (Client)** → dispositivo do usuário que acessa o site (computador, celular ou navegador).
* **Servidor (Server)** → local onde os arquivos do site estão armazenados.

Quando o usuário acessa um site, o cliente envia uma **requisição** ao servidor, 
que responde enviando os arquivos da página.

---

## 🧩 Composição de um Website

Um site normalmente é composto por três tecnologias principais:

* **HTML**
* **CSS**
* **JavaScript**

Cada uma possui uma função específica.

---

## 📰 Comparação com uma equipe de criação

O professor utilizou uma analogia para explicar a função de cada tecnologia:

* **Jornalista → HTML**
  Responsável pelo **conteúdo** da página.

* **Designer → CSS**
  Responsável pelo **estilo e aparência**.

* **Programador → JavaScript**
  Responsável pela **interação e comportamento** do site.

---

## ⚠️ HTML e CSS não são linguagens de programação

Apesar de serem essenciais para a criação de páginas web:

* **HTML** é uma **linguagem de marcação**.
* **CSS** é uma **linguagem de estilização**.

Por isso, não são consideradas linguagens de programação.

---

## 💻 JavaScript

O **JavaScript** é uma **linguagem de programação** utilizada para criar interatividade nas páginas web.

Com JavaScript é possível:

* responder ações do usuário
* alterar elementos da página
* criar animações
* tornar os sites mais dinâmicos

---

## ✅ Conclusão

A aula apresentou os conceitos iniciais sobre o funcionamento da web e explicou o papel do JavaScript dentro da criação de páginas. 
Também foi possível entender como HTML, CSS e JavaScript trabalham juntos para formar um site.
